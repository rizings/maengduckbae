프리미티브타입 : 원시타입변수

변수는 용도와 크기가 정해져있다.
int -> 정수를 저장하기위한 4byte크기의 자료형
1bit = 0 or 1 (이진수표현단위)
1byte = 8bit
1byte는 2^8개 만큼의 숫자를 표현할수있음
2byte는 2^16개 만큼의 숫자를  표현할수있음
4byte는 2^32개 만큼의 숫자를 표현할수있음

double -> 실수를 저장하기위한 8byte 크기의 자료형
지수와 가수부로 숫자를 표현
10^100 -> 10은 가수부, 100은 지수부로 표현

비트의 가장 앞자리는 부호를 표현하는데에 사용함.
0 -> 양수
1 -> 음수

컴퓨터는 음수를 표현하기위해 2의보수표현을 사용함
2의보수 : 이진수를  반전해서 1더함.
ex)
4의 이진수 : 00000100
4의 2의보수 : 11111011 + 1
-4의 이진수 : 11111100

char는 문자를 저장하기위한 2byte 자료형
0~127까지는 영어와 특수문자를 표현하기위한 아스키코드로 사용됨
127보다 큰 숫자들은 외국어 및 기타 특수문자를 표현하기위한 유니코드로 사용됨

리터럴 : 값을 저장하는데 사용하는 상수표현
int a = 123 // 123은 int의 리터럴
long b = 1234L // 1234L은 long의 리터럴
double c = 1.10 // 1.10은 double의 리터럴
float f = 1.10f // 1.10f는 float의 리터럴

String은 문자열을 저장하기위한 레퍼런스 타입 자료형
문자열에 +연산을 사용하면 붙여주기
문자열 + ? => 문자열?
? 자리에는 변수든, 리터럴이든 올수있음.

객체와 클래스
객체 : 실존하는 모든 대상 -> 메모리에 올려진 모든 대상.
        실제로 사용하는 대상
        객체, 인스턴스, Object라고 부름.

클래스 : 존재하거나 만들어야할 객체의 속성과 기능을 일반화, 추상화로 정의한 틀.
클래스구성
속성 : 객체의 상태를 저장하는 변수.
        필드, 멤버변수라고도 부름.

기능 : 객체가 할수있는 행위(메소드)
        메소드, 함수라고 부름

객체생성방법
new 연산자를 이용하여 객체생성
ex) Account acc = new Account();
객체생성 : 객체를 만들어서 메모리에 올리는 행위. 
              컴퓨터는 메모리에 올려진 자원만 만질수있음.

acc.deposit(1000); //객체생성이후 객체는 .연산자를 이용해 속성과 기능이 사용가능해짐.

생성자 : 객체생성때 사용되는 함수같은 것.
           객채생성시 반드시 호출되어야함.
	매개변수가 있는 생성자를 작성할수 있음.
ex) Account acc = new Account("1234-1234","홍길동","1234", 10000);
일반적으로 매개변수가 있는 생성자를 이용해 속성들을 초기화 함.

클래스내부에 생성자가 하나도 없을시 컴파일러가 자동으로 기본(default) 생성자를 삽입한다

class GoodsStcok
{ // 내부에 생성자 없음 
}
라고 작성하면 아래와 같이 변경함.
class GoodsStock
{
	GoodsStock() { }  //기본생성자 삽입
}

this( ) : 생성자 내부에서 다른 생성자를 호출할때 사용한다.
중복코드가 많은 생성자를 개선할수있다.
this.멤버변수 : 매개변수하고 이름이 같을때 멤버변수와 매개변수를 구분하기위한 용도로 사용.
생성자 오버로딩 : 시그니처(매개변수들의 갯수, 종류, 순서)가 다르면 여러번 중복할수있음.
생성자 오버로딩은 다형성의 한 예.
다형성 : 여러가지 작업을 하나의 개념으로 처리하는 것.

현재까지 배운 객체지향 특성
다형성, 은닉화, 상속, 캡슐화.

인위적 익셉션(예외) 발생하기
throw new Exception()을 이용하면 인위적인 예외를 발생시킬수있다
인위적인 예외를 통해서 일관성 있는 예외처리를 수행할수있다.

퀴즈 : Car 클래스를 작성하시오
속성으로 속도(int, private), 차종(String, private)을 가진다
속도는 0~ 200 사이의 값만 가질수있다.
속성들은 생성자를 통해 초기화한다.
 - 생성자에 값이 전달안될시 속도는 0으로 초기화.
 - 차종은 무조건 입력되어야한다.

기능으로 가속과 브레이크가 있다
가속은 한번 호출될때 마다 속도를 10증가시킨다.
브레이크는 한번호출될때 마다 속도를 20감소시킨다.

퀴즈 : 사각형, 삼각형 클래스를 아래와 같은 정보로 작성하시오.

사각형클래스(Rectangle)
 - 속성으로 w, h를 가짐
 - 속성은 생성자를 이용해 초기화 할것
 - 생성자의 호출은 아래와 같은 형식으로 호출됨
 ex) 
 new Rectangle(); //전달되지 않은 정보는 1로 초기화 할것.
 new Rectangle(10); //전달되지 않은 정보는 1로 초기화 할것.
 new Rectangle(10, 10);
 - 넓이를 구하는 getArea()가 있음

삼각형클래스(Triangle)
 - 속성으로 w, h를 가짐
 - 속성은 생성자를 이용해 초기화 할것
 - 생성자의 호출은 아래와 같은 형식으로 호출됨
 ex) 
 new Triangle(); //전달되지 않은 정보는 1로 초기화 할것.
 new Triangle(10); //전달되지 않은 정보는 1로 초기화 할것.
 new Triangle(10, 10);
 - 넓이를 구하는 getArea()가 있음
